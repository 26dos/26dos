### About

I work at the intersection of **autonomous agents and on-chain economics** —
how should an LLM-driven agent reason about a leveraged DeFi position? What
record should it leave behind? Which patterns of speculative trading repeat
themselves across chains?

Most of what I publish is research-flavored tooling: things I built to
answer my own questions, then cleaned up enough that someone else might
find them useful too.

### Currently

- 🔭 Building tooling for studying speculative DeFi (memecoins, prediction markets, perps)
- 🧪 Tool-using LLM agents for DeFi position risk analysis
- 📚 Reading too much about MEV microstructure
- 🌏 Based in Singapore, talk a lot about agents at the NUS Crypto Reading Group

### Open to

Collaborations on:
- agent accountability / verifiable inference for on-chain agents
- empirical MEV / market microstructure
- speculative-DeFi datasets and shared benchmarks
